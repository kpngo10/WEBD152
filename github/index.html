Getting Started

You do have to know a bit of command line not just to set up the website but also to keep uploading files for class.

The General Steps

  1. Create an account
  2. Create new repository
  3. Clone the repository
  4. Go to repository's settings
  5. Go down to pages
  6. Make sure the source for Build and Deployment is
    a) deploy from a branch and
    b) under branch, main is selected and your folder is /(root)
  7. Wait a bit
  8. Refresh the page
  9. Your repository pages settings should say
    "Your site is live at https://username.github.io/repositoryname"

The Specifics

Continuing from step 3: cloning the repository

  1. Open command line

    a) Windows: Press Windows Key + R, type in cmd.exe, press Enter.
    b) MacOS: Press ⌘ + Spacebar, or Command + Spacebar, type in Terminal. Press Enter.

  2. Navigate to where you want the webd152 folder to be created using commands

    a) Most commonly used:
    • pwd: print working directory
    • cd: change directory
    • ls: list files in a directory
    • mkdir: create new directory
    • rm: remove a file, add -r to remove a directory
    • touch: create a file
    • open: open a file
    • code: open a file in Visuaal Studio Code(?)
    b) Example: I want my wedb152 folder to be in MESA folder, it would look like
    cd Desktop/MESA/SM2024, you just need the command + directory path.
  3. To clone a directory:
    a) go to your webd152 repository's settings
    b) click on Code
    c) copy the local SSH link, it should look like
    git@github.com:username/webd152.git
    d) back to your command line / terminal
    e) type in the command: git clone [your local SSH link you just copied]
    so it would look like: git clone git@github.com:kpngo10/webd152.git
    f) folder called webd152 should be created in where your current directory is
